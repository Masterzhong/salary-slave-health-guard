# 牛马健康助手 🐂🐴

> 打工人の健康守护神，守护你的每一根头发！

## 项目简介

牛马健康助手是一款专为程序员/上班族设计的健康提醒工具，帮助你在繁忙的工作中保持健康。

## 功能特点

### 💧 智能饮水提醒
- 可自定义提醒间隔（默认40分钟）
- 工作时间内自动提醒
- 幽默风趣的文案让你会心一笑

### 🏃 久坐站立提醒
- 每60分钟提醒一次
- 保护你的腰椎和菊花

### 📥 系统托盘化
- 最小化到系统托盘
- 右键菜单快速操作
- 不占用任务栏空间

### 🔛 开机自启动
- 通过Windows注册表实现
- 开机自动运行
- 电脑启动即开始守护

### 🔔 桌面通知
- Windows 10 原生通知
- 支持点击交互
- 实时显示提醒内容

## 环境要求

- **Python**: 3.6.8（必须兼容）
- **操作系统**: Windows 10+
- **依赖**: PyQt5 5.12.3, win10toast

## 安装

1. 克隆或下载本项目

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行程序：
```bash
python src/main.py
```

## 使用说明

### 首次运行

程序启动后会显示主界面，你可以：
- 调整饮水/久坐提醒间隔
- 开启/关闭开机自启动
- 开启/关闭桌面通知
- 点击"重置计时"按钮手动重置提醒

### 最小化到托盘

关闭主窗口后，程序会最小化到系统托盘。
- 左键点击托盘图标：显示主窗口
- 右键点击托盘图标：显示菜单

### 托盘菜单功能

- 💧 饮水提醒：立即发送饮水提醒
- 🏃 久坐提醒：立即发送久坐提醒
- 📋 打开主窗口：显示主界面
- ❌ 退出：完全退出程序

## 配置文件说明

配置文件位于 `config/settings.json`：

```json
{
    "water_interval": 40,          // 饮水提醒间隔（分钟）
    "sit_interval": 60,            // 久坐提醒间隔（分钟）
    "work_hours": {
        "start": "09:00",         // 工作开始时间
        "end": "18:00"            // 工作结束时间
    },
    "auto_start": false,           // 开机自启动
    "notifications_enabled": true  // 启用通知
}
```

## 注意事项

1. **工作时间**: 提醒只在工作时间内发送（默认9:00-18:00）
2. **通知权限**: 确保Windows通知已开启
3. **托盘运行**: 程序关闭后会最小化到托盘，而非完全退出

## 常见问题

### Q: 程序无法启动？
A: 检查Python版本是否为3.6.8，依赖是否正确安装

### Q: 通知不显示？
A: 检查Windows通知设置，确保通知已启用

### Q: 开机自启动无效？
A: 尝试以管理员权限运行程序
